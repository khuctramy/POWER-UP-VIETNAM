# INSTALLATION
## Step 1: Clone my source code
- `git clone https://github.com/khuctramy/teaching-app.git`
## Step 2: Access the project directory
- `cd expo-learningApp`
## Step 3: Install dependencies
- `yarn install`
## Step 4: Run App
- `yarn start`
